Token-Level Pipeline Parallelism

This project aims to perform a token-level pipeline parallelized model of training an LLM. The token-level pipelining will also be a useful method to process greater chunks of text, thus enabling the possibility of more attention on specific words in a passage. We will use tools such as NeAt vision, which map attention heads, to identify and compare changes in attention values and mappings between a traditionally trained LLM and our LLM.
